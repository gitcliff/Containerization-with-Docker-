# ------------------------------  
# üß© SECTION TITLE: Download MongoDB and Mongo Express Docker Images  
# ------------------------------  
docker pull mongo  
# ‚úÖ What it does: Downloads the latest official MongoDB Docker image  
# üß† Why it matters: MongoDB is the database your app needs; pulling the image prepares it for container creation  

docker pull mongo-express  
# ‚úÖ What it does: Downloads the latest official Mongo Express image (a web UI for MongoDB)  
# üß† Why it matters: Provides a convenient web interface to interact with your MongoDB instance without needing CLI tools

# ‚úÖ What it does: Lists all Docker images currently downloaded on your system  
# üß† Why it matters: Lets you verify that the mongo and mongo-express images were successfully pulled and are available to run containers  

# ------------------------------  
# ‚ñ∂Ô∏è HOW TO RUN / EXECUTE THESE COMMANDS  
# ------------------------------  
# Run each command in your terminal (bash or PowerShell). Pull commands will download images if not cached.  

# ------------------------------  
# ‚úÖ NOTES / GOTCHAS / TIPS  
# ------------------------------  
# - `docker pull mongo` pulls the latest stable MongoDB image by default.  
# - You can specify versions like `mongo:5.0` to pull a specific MongoDB version.  
# - Mongo Express requires a running MongoDB container to connect to.  
# - Use `docker run` commands after this to start containers based on these images.
